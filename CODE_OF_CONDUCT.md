# Contributor Covenant Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) v2.1.

In short: be respectful, be welcoming, assume good faith, and keep discussion
focused on the work. Harassment of any kind is not tolerated.

Report unacceptable behaviour to **support@batchnepal.com**. All reports are
reviewed and handled confidentially.
